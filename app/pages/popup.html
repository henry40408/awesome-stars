Why a token?

  • NO permission needs to be granted to the token.
  • According to GitHub Documentation, only 60 requests is allowed to send in an hour, and it is not enough for getting star count for all repositories in a single awesome curated list.
  • With a token, the extension can send 5,000 requests instead. For better performance and user experience, the extension would also cache the result.
